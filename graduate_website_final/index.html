使用指南

如何借书？

  1. 在主菜单选择"自助借书"
  2. 按照屏幕提示刷卡识别身份
  3. 扫描书籍条形码
  4. 确认借阅信息

如何还书？

  1. 在主菜单选择"自助还书"
  2. 扫描书籍条形码
  3. 将书籍放入还书口
  4. 等待系统确认
